# cloudflare-java-client
A Java/Groovy client for the CloudFlare v4 API

Currently supports basic CRUD operations on zones and records.

To use (in Gradle):

```groovy
repositories {
    jcenter()
}

dependencies {
    compile 'com.mhackner:cloudflare-java-client:0.1'
}
```
